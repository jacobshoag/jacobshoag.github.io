About Dyslex-Kriyah

Dyslex-Kriyah is designed to make Hebrew reading easier for individuals with dyslexia. The font incorporates features like heavier bottoms, distinctive shapes, and larger openings to improve readability.

Principles Behind Dyslex-Kriyah

1. Heavier Bottoms

One of the key principles behind the Dyslex-Kriyah font is giving letters heavier bottoms. This design choice ensures that letters have a consistent baseline, reducing the chances of dyslexic readers rotating or inverting them. The added visual weight at the bottom makes it easier for readers to maintain orientation, enhancing readability.

2. Distinctive Shapes

Dyslex-Kriyah tries to make each letter distinct whenever possible. The goal is to help dyslexic readers avoid accidentally swapping one letter for another.

3. Distinctive Slants

When letter shapes are similar, Dyslex-Kriyah differentiates them by using unique slants. This makes it easier to quickly tell one letter from another.

4. Wider Spacing

The font uses wider spacing between letters and words to prevent crowding and make reading easier.
